Instructions for the program - mosaic.py

Python is needed to run the program so ensure that is done, after python is 
installed the following command must be run inorder to use the program:

pip install Pillow

now you can run the program by typing the following command

python mosaic.py <name of target image> <folder of images to be used>

the program will create a mosaic out the images in the folder and display 
it using the Python imaging library

variables can be changed in the program in any given text editor
